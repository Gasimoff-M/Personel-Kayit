# Personel Kayıt Uygulaması

Bu proje, kurum ve işletmelerde personel bilgilerinin düzenli bir şekilde tutulmasını sağlamak amacıyla geliştirilmiş bir Personel Kayıt Uygulamasıdır.
Uygulama sayesinde personel ekleme, güncelleme, silme ve listeleme işlemleri kolayca gerçekleştirilebilir.

# Özellikler

* Personel ekleme, silme ve güncelleme
* Personel listeleme ve arama
* Departman ve pozisyon bilgileri
* Kullanıcı dostu arayüz
* Veritabanı destekli kayıt sistemi
